Data types can be split into 2.
Primitive and Non-Primitive

Primitive data types are
Numbers
Strings
Booleans
Null
Undefined
Symbol

Non-Primitive data types are
Objects
Arrays

Primitive data types are NON-Modifiable. Once it is created we cant change it example
let word = 'JavaScript'

trying to modify the string that stores word, should throw and error. Any datatype under a Single Quote, Double Quote, or Backtick Quote is a String.
word[0] = 'Y'
